# Malang VR Tourism

Virtual reality application about Malang city tourism destination.


## Getting Started

### Installing

```
npm install
```

### Running aplication

```
npm run start
```

### Build for production

```
npm run bundle
```

## Built With

* [React360](https://facebook.github.io/react-360/) - The vr framework used
